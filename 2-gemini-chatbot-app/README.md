# Gemini Chatbot App

Fullstack Gemini LLM Project built with Node.js

## API

URL: `/chat`

Body example:
```json
{
    "success": true,
    "data": "The capital of England is **London**.",
    "message": "Berhasil ditanggapi oleh Google Gemini Flash!"
}
```
